🎓 Cosa sta succedendo?

Il server Flask riceve la tua stringa tramite l'URL e la inserisce direttamente nel codice HTML della pagina.

Il problema nel codice:

// In index.html
{{ "{{" }} q | safe {{ "}}" }}

L'uso del filtro | safe dice a Flask: "Non pulire i caratteri speciali, mi fido di questo input". Questo permette a tag come <script> di essere eseguiti dal browser.

Come risolverlo:

Rimuovi semplicemente il filtro | safe. Flask userà il suo sistema di Auto-Escaping predefinito:

{{ "{{" }} q {{ "}}" }}

In questo modo, il browser riceverà &lt;script&gt; invece di <script>, visualizzando il codice come semplice testo senza eseguirlo.
