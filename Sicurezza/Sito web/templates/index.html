🎓 Cosa sta succedendo? (SQL Injection)

Il server prende esattamente ciò che scrivi nella barra di ricerca e lo "incolla" direttamente in un comando SQL per il database.

Il problema nel codice (app.py):

sql_statement = f"SELECT * FROM users WHERE name = '{search_query}'"

Prova ad hackerarlo!

  • Il Bypass dell'autenticazione: Invece di cercare un nome, prova a digitare questo payload esatto nella barra di ricerca:
    ' OR '1'='1
  • Cosa succede: La query diventa SELECT * FROM users WHERE name = '' OR '1'='1'. Poiché 1 è sempre uguale a 1, il database restituirà tutti gli utenti, inclusa la Flag dell'Amministratore!
  • L'Errore: Prova a inserire solo un singolo apice: ' per mandare in crash la sintassi SQL e vedere l'errore nudo e crudo.
